# DAR

D AR Archive Parser.

A library and utility.

## Build

* Debug build:

> Creates `./dar` executable.

```
dub build
```

* Release build:

> Creates `./dar` executable.

```
dub build --build=release
```

* Library build (no executable)

> Creates `libdar.a`.

```
dub build --config=library
```

## Test

```
dub test
```
